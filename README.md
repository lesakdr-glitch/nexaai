# ✨ Nexa AI

Продвинутый AI-чат для помощи с Minecraft Fabric модификациями и Nexa DLC клиентом.

## 🎯 Особенности

- 🎨 **VisionOS + Glassmorphism** дизайн с тёмной темой
- 💬 **Полнофункциональный чат** с поддержкой markdown и блоков кода
- 📝 **История чатов** с автосохранением в localStorage
- ✏️ **Редактирование названий** чатов (двойной клик)
- 🗑️ **Удаление чатов** с подтверждением
- ⌨️ **Ctrl+Enter** для отправки сообщений
- 🤖 **6+ моделей AI** на выбор
- 📱 **Адаптивный дизайн** (мобильный + десктоп)
- ✨ **Плавные анимации** Framer Motion
- 🎯 **Приветственное сообщение** при новом чате
- 🔑 **Модальное окно API ключа** при первом запуске
- 💡 **Подсветка кода** с Highlight.js (Atom One Dark)
- 📋 **Копирование блоков кода** одной кнопкой

## 🛠️ Технологический стек

- **Next.js 14** - Framework
- **TypeScript** - Язык программирования
- **Tailwind CSS** - Стили
- **Framer Motion** - Анимации и переходы
- **Lucide React** - Иконки
- **Zustand** - Управление состоянием
- **React Markdown** - Отрисовка markdown
- **Highlight.js** - Подсветка синтаксиса кода

## 🚀 Установка и запуск

```bash
# Клонируем проект
git clone <repo>
cd nexa-ai

# Устанавливаем зависимости
npm install

# Запускаем dev сервер
npm run dev

# Открываем в браузере
# http://localhost:3000
```

## 📋 Использование

1. **Первый запуск**: Введите API ключ для codex.sale в модальном окне
2. **Создание чата**: Нажмите "Новый чат" в сайдбаре
3. **Отправка сообщений**: Напишите вопрос и нажмите отправить или Ctrl+Enter
4. **Редактирование названия**: Двойной клик по названию чата в сайдбаре
5. **Удаление чата**: Наведите на чат и нажмите иконку корзины
6. **Смена модели**: Выберите модель в выпадающем списке в сайдбаре
7. **Обновление ключа**: Нажмите "Сменить ключ" в низу сайдбара

- **Ctrl+Enter** - Отправить сообщение

## 🔐 Безопасность

- API ключ сохраняется только в localStorage браузера
- Ключ никогда не отправляется на сторонние сервера
- Все данные хранятся локально
